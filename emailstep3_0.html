To create a list, you need to have new or existing contacts, leads or referals to include in a group, or in more than one group, or rather lists.

To create your list, the following steps should be taken:
  1. On the menu panel, from the 'Audience' dro-down list, select 'Lists ' Your lists will appear under 'My lists'.
  2. You can edit an existing list, or create a new list.
  3. To create a new list select 'Create list '
  4. Edit the the following list specifications: Identity, Contact Information and Settings, then save.
  5. Once saved, the list will be accessible from the lists, from where additional settings and functions can be added.
